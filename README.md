# 🚗 Automobile Sales Dashboard

## 📌 Overview

An interactive dashboard built with **Dash** to explore automobile sales trends across years and recession periods. It enables quick insights into sales behavior, category performance, and economic impact through dynamic visualizations.

---

## ⚙️ Tech Stack

* Python
* Dash
* Plotly
* Pandas

---

## 📊 Key Features

* 📅 Toggle between **Yearly** and **Recession** analysis
* 🎛️ Interactive dropdown filters
* 📈 Real-time updating charts
* 📊 Clean, responsive layout

---

## 📉 Visual Insights

### 🔴 Recession Analysis

* Sales trend over recession years (line chart)
* Sales by vehicle type (bar chart)
* Ad expenditure share (pie chart)
* Impact of unemployment on sales (bar chart)

### 🔵 Yearly Analysis

* Overall yearly sales trend (line chart)
* Monthly sales pattern (line chart)
* Average sales for selected year (bar chart)
* Ad expenditure by vehicle type (pie chart)

---

## 🔁 Interactivity

* Select report type using dropdown
* Year selection enabled only for yearly view
* Charts update dynamically based on inputs

---

```bash
pip install dash pandas plotly more-itertools
```
